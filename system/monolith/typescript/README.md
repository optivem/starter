# Monolith (TypeScript)

This is a sample monolithic application written in TypeScript using Next.js with server-side rendering.

## Instructions

Ensure you have Node.js 22 installed

```shell
node --version
```

Install dependencies

```shell
npm install
```

Build the application

```shell
npm run build
```

Run the application in development mode

```shell
npm run dev
```

Run the application in production mode

```shell
npm run build && npm start
```

App should now be running on:
http://localhost:3000/
