# Backend (.NET)

This is a sample backend API service written in C# using ASP.NET Core.

## Instructions

Open up the 'backend' folder

```shell
cd backend
```

Ensure you have .NET SDK 8.0 installed

```shell
dotnet --version
```

Build the application

```shell
dotnet build
```

Run the application

```shell
dotnet run
```

App should now be running on:
http://localhost:8081/
