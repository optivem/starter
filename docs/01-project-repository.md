# Project Repository

## 1. Create System Repository

Create the System Repository you decided on in the [Prerequisites](00-prerequisites.md) step.

1. Repository Name: Use the repo name you chose in Prerequisites (e.g. `acme-shop`).
2. Choose Visibility: Public
3. Add Readme: On
4. Add License: MIT License (or another Open Source license)

## 2. Set Up README

Add a README with the project name, license name, and a Contributors section.

## Checklist

1. Repository is publicly accessible
2. README includes the project name, license name, and a Contributors section
3. Repository includes a LICENSE file
